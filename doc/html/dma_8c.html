void DMA_init ( void  )

This is an initialization place holder for:

  1. Enabling DMA MUX clock PCC_PCCn[PCC_DMAMUX_INDEX] (not needed when START bit used).
  2. Enabling desired channels by setting ERQ bit (not needed when START bit used).

TCD0: Transfers string to a single memory location

Source Address: TCD0_Source (11 byte array).
Increment source address by 1 byte for each transfer
Decrement source address by 11 after major loop.

Destination Address: TCD0_Dest volatile char.
Do not add offset to destination address after minor loop.
Write 1 byte at a time.
Do not adjust address after major loop.
