eDMA Linking Channel

Description

Summary: The intent of this example is to illustrate how to implement the Linking Channel feature of the eDMA
The Linking Channel Feature of the DMA where you can set up a link so when a channel finishes a major or minor loop it
activates another DMA channel to start working with its own configuration depending on what you desire to do with it.
This function is better appreciated in the following image.


In this example we use this feature to link CH0 to CH1 after CH0's minor loops to copy the phrase "Hello World" from a string
defined inside the dma driver and sent it to another string (also defined in the dma driver) but here we set the DMA CH0 to
send the first 6 bytes of the source string ("Hello ") and after each minor loop of the channel the CH1 starts sending
the rest of bytes left in the original array ("World").

Even when here we use the Linking Channel feature to finish sending an array you have to know that this isn't its only
possible application, for example: as the scatter gather, the linking channel can be used to generate a state machine
sub-system where instead of having 2 TCD configurations of a channel you set up 2 channels and link them so they work one
after the other.

Design

  • Disable Watchdog.
  • Initialize system clocks and Run mode.
  • Configure DMA requests to be always available.
  • Set up DMA TCD configurations:
    -Set up DMA TCD CH0 to transfer the first 6 bytes of the source array.
    -Enable Linking Channel from CH0 to CH1.
    -Set up DMA TCD CH1 to transfer the 5 bytes left of the source array.
  • Enable DMA Channel Requests and interruptions.

#include "device_registers.h"
#include "clocks_and_modes.h"
#include "dma.h"
void WDOG_disable (void)
{
WDOG->CNT=0xD928C520; /* Unlock watchdog */
WDOG->TOVAL=0x0000FFFF; /* Maximum timeout value */
WDOG->CS = 0x00002100; /* Disable watchdog */
}
int main(void) {
WDOG_disable(); /* Disable WDOG */
SOSC_init_8MHz(); /* Initialize system oscillator for 8 MHz xtal */
SPLL_init_160MHz(); /* Initialize SPLL to 160 MHz with 8 MHz SOSC */
NormalRUNmode_80MHz(); /* Init clocks: 80 MHz sysclk & core, 40 MHz bus, 20 MHz flash */
DMAMUX_LC_init(); /* Initialize DMAMUX to always generate DMA requests for DMA CH0 and CH1 */
DMA_TCD_LC_Config(); /* Set up DMA TCD CH0 and CH1 to work with linking channel to completely 'save' "Hello World" */
DMA->SERQ |= DMA_SERQ_SERQ(0); /* Enable DMA CH0 request */
DMA->SERQ |= DMA_SERQ_SERQ(1); /* Enable DMA CH1 request */
S32_NVIC->ICPR[1/32] |= 1 << (1 % 32); /* DMA CH1: clr any pending IRQ */
S32_NVIC->ISER[1/32] |= 1 << (1 % 32); /* DMA CH1: enable IRQ */
for (;;) {
/* Wait forever */
/* Look for the TCD_LC_Dest array inside the dma driver */
}
return 0;
}
void DMA1_IRQHandler (void)
{
DMA->CDNE |= DMA_CDNE_CADN_MASK; /* Clear Done status bit */
DMA->CINT |= DMA_CINT_CINT(1); /* Clear Interrupt request */
}
