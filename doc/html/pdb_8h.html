PDB is a timer used to trigger other peripherals, in this case the ADC
For this example, the PDB will be used in continuous mode and triggered by software
When the counter overflows, it will start to count again, and it will trigger the ADC when it reaches the CH0_DLY(1000)
The period is of 9ms
