void SOSC_init_8MHz ( void  )

SOSC Initialization (8 MHz):

This function initializes system oscillator (SOSC) for 8MHz crystal,
configures SOSCDIV1 and SOSCDIV2 to 8MHz, (divided by 1),
ensures SOSC control and status reg. is unlocked, and wait for SOSC to be valid.

◆ SPLL_init_160MHz()

void SPLL_init_160MHz ( void  )

SPLL Initialization (160 MHz):

This function initializes system PLL (SPLL) to 160 MHz using 8MHz SOSC,
SPLLDIV1 is divided by 2, and SPLLDIV2 is divided by 4,
PREDIV = 0, and MULT = 24, in SPLL config reg:
SPLL_CLK = 8MHz / (PREDIV+1) * 40 / 2 = 160 MHz.
ensures SPLL control and status reg. is unlocked, and wait for SPLL to be valid.
