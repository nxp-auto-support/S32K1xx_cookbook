ADC Flexible Scan

Description

Summary: This example intends to show how to combine ADC, DMA and PDB to implement an ADC flexible storage.

In this project, PDB triggers ADC0 CH0 measurements which will be saved inside an internal memory buffer through DMA,
this way the MCU doesn't need to read the ADC result register because the transfers will be done by DMA.

In order to achieve the desired behaviour we have to set up different DMA Channels to make different tasks, first we have
to make a channel to transfer data from the ADC result register to a memory buffer after an ADC COCO trigger, when this data
is transfered we have to call another DMA channel to change the ADC Channel we are currently reading, so this DMA will transfer
information from an internal buffer containing the desired channels to the ADC0 SC1_A register. Thanks to this transfer, when
the next ADC HW trigger happens it will measure the channel defined by the DMA and once the ADC conversion has completed all of
this will happen again.

To be able to make this we have to use the Linking Channel feature of the DMA to link Channel 0 to Channel 1 after each minor
loop and in this case the major loop will also link channel 0 to channel 1. When every minor loop of channel 0 ends, this
will trigger Channel 1 which will proceed to complete a minor loop.


In this example we will use 3 different ADC Channels to take 4 different measures, so we have a major loop of 12 minor loops
for the DMA Channel which saves the ADC results and a major loop containing 3 minor loop for the DMA channel that changes the
ADC Channel to be measured. Once all 12 measures have been saved in the internal memory buffer, measurement of the ADC
disabled.


Design

  • Disable Watchdog.
  • Initialize system clocks and Run mode.
  • Initialize ADC0 CH0 with HW trigger and DMA Requests
  • Configure DMA to accept requests from ADC0.
  • Set up DMA TCD configurations:
    -DMA TCD CH0:
    -Set up to have 12 minor loops per major loop.
    -Source Addres: ADC0 Result Register.
    -Destination Addres: Array to save the results.
    -Set linking channel after each minor loop to CH1.
    -Enable interrupt after major loop.
    -DMA TCD CH1:
    -Set up to have 3 minor loops per major loop.
    -Source Addres: Array which contains the ADC Channel number.
    -Destination Address: ADC0 SC1_A Register.
  • Enable DMA Channel Requests and interruptions.
  • Set up PDB to trigger ADC every second

#include "device_registers.h"
#include "dma.h"
#include "clocks_and_modes.h"
#include "pdb.h"
#include "ADC.h"
void WDOG_disable (void)
{
WDOG->CNT=0xD928C520; /* Unlock watchdog */
WDOG->TOVAL=0x0000FFFF; /* Maximum timeout value */
WDOG->CS = 0x00002100; /* Disable watchdog */
}
int main(void)
{
WDOG_disable(); /* Disable WDOG */
SOSC_init_8MHz(); /* Initialize system oscillator for 8 MHz xtal */
SPLL_init_160MHz(); /* Initialize SPLL to 160 MHz with 8 MHz SOSC */
NormalRUNmode_80MHz(); /* Init clocks: 80 MHz sysclk & core, 40 MHz bus, 20 MHz flash */
ADC_FlexScan_Config(); /* Initialize ADC0 CH0 with HW Trigger and DMA Request */
DMAMUX_FlexScan_init(); /* Initialize DMA to take requests from ADC0 */
DMA_TCD_FlexScan_Config(); /* Set up TCD CH0 to save measurements from ADC0 and link to CH1 to change ADC0 channel to measure */
DMA->SERQ = DMA_SERQ_SERQ(0); /* Enable Requests for DMA Channel 0 */
PDB_FlexScan_Config(); /* Configure PDB to trigger ADC0 every second */
S32_NVIC->ISER[0/32] |= 1<<(0%32); /* Enable interruption for DMA CH0 */
for(;;) {
}
return 0;
}
void DMA0_IRQHandler (void) {
DMA->CDNE = DMA_CDNE_CDNE(0); /* Clear Done Status Flag of DMA Channel 0 */
DMA->CINT = DMA_CINT_CINT(0); /* Clear Interruption request flag of DMA Channel 0 */
PDB0->SC &=~ PDB_SC_PDBEN_MASK; /* Turn off PDB to stop getting samples after DMA CH0 major loop */
}
