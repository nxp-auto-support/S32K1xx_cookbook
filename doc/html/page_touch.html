Touch-Sensing

Description

Summary: The intent of this example is to illustrate how to detect touch events by using the Charge Re-distribution method..

  1. Configure touch button (and EGS) electrodes floating.
  2. Distribute Electrode and Cext charge.
  3. Delay to distribute charge.
  4. Start Cext voltage ADC conversion.
  5. Redistribute Electrode and Cext charge.
  6. Equivalent voltage digitalization (Poll conversion complete flag).
  7. Compare with the Touched reference.
  8. LED action.

    Design

The ADC configuration was taken from the ADC-Cookbook Example.
The Charge distribution and Charge redistribution steps are implemented with inline functions.
The corresponding channels are read and the results are compared with the given limits to determine if the pad's were touched.

  • Blue led follows the SW7 pad.
  • Green led follows the SW8 pad.

    There are two configurable params:
  • ELEC_TOUCHED_LIMIT_VALUE: ADC result from which the pad is considered touched.
  • ELEC_DIST_CHARGE_DELAY: SW delay to distribute charge into the external capacitor.

#define PTD0 (0) /* Port D0: FRDM EVB output to blue LED */
#define PTD15 (15) /* Port D15: FRDM EVB output to red LED */
#define PTD16 (16) /* Port D16: FRDM EVB output to green LED */

/*****************************************************************************
*
* Function: void PORT_init (void)
* Description: Configure as output the corresponding GPIO's for the RGB led.
*
*****************************************************************************/
void PORT_init (void)
{
PCC->PCCn[PCC_PORTD_INDEX ]|=PCC_PCCn_CGC_MASK; /* Enable clock for PORTD */
PORTD->PCR[PTD0] = PORT_PCR_MUX(1); /* Port D0: MUX = GPIO */
PORTD->PCR[PTD15] = PORT_PCR_MUX(1); /* Port D15: MUX = GPIO */
PORTD->PCR[PTD16] = PORT_PCR_MUX(1); /* Port D16: MUX = GPIO */
PTD ->PDDR |= 1<<PTD0 /* Port D0: Data Direction = output */
|1<<PTD15 /* Port D15: Data Direction = output */
|1<<PTD16; /* Port D16: Data Direction = output */
/* Turn-off RGB led */
PTD ->PSOR |= 1<<PTD0 /* Port D0: Data Direction = output */
|1<<PTD15 /* Port D15: Data Direction = output */
|1<<PTD16; /* Port D16: Data Direction = output */
}
/*****************************************************************************
*
* Function: void WDOG_disable (void)
* Description: Disable the Watchdog timer.
*
*****************************************************************************/
void WDOG_disable (void)
{
WDOG->CNT=0xD928C520; /* Unlock watchdog */
WDOG->TOVAL=0x0000FFFF; /* Maximum timeout value */
WDOG->CS = 0x00002100; /* Disable watchdog */
}
/*****************************************************************************
*
* Function: int main(void)
* Description: Sense touch pads
*
*****************************************************************************/
int main(void)
{
WDOG_disable(); /* Disable WDOG */
SOSC_init_8MHz(); /* Initialize system oscillator for 8 MHz xtal */
SPLL_init_160MHz(); /* Initialize SPLL to 160 MHz with 8 MHz SOSC */
NormalRUNmode_80MHz(); /* Init clocks: 80 MHz sysclk & core, 40 MHz bus, 20 MHz flash*/
PORT_init(); /* Init port clocks and gpio outputs */
ADC_init(); /* Init ADC resolution 12 bit */
/* Enable clock for PORTC */
PCC->PCCn[PCC_PORTC_INDEX ]|=PCC_PCCn_CGC_MASK;
/* Endless loop */
for(;;)
{
/* If PAD1 is touched, Blue led turns on */
if(TouchSense1())
{
PTD ->PCOR |= 1 << PTD0;
}
else
{
PTD ->PSOR |= 1 << PTD0;
}
/* If PAD2 is touched, Green led turns on */
if(TouchSense2())
{
PTD ->PCOR |= 1 << PTD16;
}
else
{
PTD ->PSOR |= 1 << PTD16;
}
}
return 0;
}
/*****************************************************************************
*
* Function: bool TouchSense1 (void)
* Description: Sense pad1
*
*****************************************************************************/
bool TouchSense1 (void)
{
bool Touched = false;
uint16_t adcResult;
uint32_t chargeDistributionPeriodTmp = ELEC1_DIST_CHARGE_DELAY;
// Configure touch button (and EGS) electrodes floating
ElectrodeFloat1();
// Distribute Electrode and Cext charge
ChargeDistribution1();
// Delay to distribute charge
while (chargeDistributionPeriodTmp) {chargeDistributionPeriodTmp--;}
// Start Cext voltage ADC conversion
convertAdcChan(ELEC1_ADC_CHANNEL);
// Redistribute Electrode and Cext charge
ChargeRedistribution1();
// Equivalent voltage digitalization
// Wait for conversion complete flag
while(adc_complete()==0){}
adcResult = read_adc_chx();
(void)adcResult;
// Compare with the Touched reference.
if(read_adc_chx() < ELEC1_TOUCHED_LIMIT_VALUE)
{
Touched = true;
}
return Touched;
}
