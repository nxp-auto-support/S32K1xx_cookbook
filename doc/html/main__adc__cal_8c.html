◆ PTC6

#define PTC6   (6)

Description:

This code interactively shows how the internal calibration of the ADC module affects the reading of the result register (R). In addition to the internal calibration, there is the possibility of modifying. two registers for the Gain (UG) and the Offset (USR_OFS) of the result by the user.

Instructions: Valid User Gain values are between 0 - 1023. Valid User Offset values are between 0 - 255. Offset and Gain values are in 2's-complement format. There are negative and positive values. MSB determines the sign. Press ENTER to send the Gain and Offset value.

The UART is only used to display the ADC Result status in the terminal at 9600 baud: TeraTerm or other software.
