External Watchdog Monitor (EWM)

Description

The EWM provides an independent EWM_OUT_b signal that when asserted resets or places an external circuit into a safe mode.
The EWM_OUT_b signal is asserted upon the EWM counter time-out.
An optional external input EWM_in is provided to allow additional control of the assertion of EWM_OUT_b signal.
In this example EWM module is initialized to assert EWM_OUT_b signal after 255ms.


Design

  • Disable watchdog
  • System clocks: Initialize SOSC for 8 MHz, sysclk for 80 MHz, RUN mode for 80 MHz
  • Initialize port pins:
    • Enable clock to Port A module
    • PTA2, Configure for EWM_OUT_b
  • Initialize EWM module
    • Refresh window is [15ms - 255ms].
  • Enable WDOG_EWM Interrupt.

  • Loop:

    • Wait forever...

    * EWM module is initialized to assert EWM_OUT_b signal after 255ms,

    #include "device_registers.h"
    #include "clocks_and_modes.h"
    #include "ewm.h"
    void PORT_init (void)
    {
    /* Pins definitions
    * ====================================
    *
    * Pin number | Function
    * ----------------- |------------------
    * PTA2 | [EWM_OUT_b]
    */
    PCC->PCCn[PCC_PORTA_INDEX] = PCC_PCCn_CGC_MASK; /* Enable clock for PORT A */
    PORTA->PCR[2] |= PORT_PCR_MUX(4); /* Port A2: MUX = ALT4, EWM_OUT_b */
    /* External Watchdog Monitor Output */
    }
    void WDOG_disable (void)
    {
    WDOG->CNT=0xD928C520; /* Unlock watchdog */
    WDOG->TOVAL=0x0000FFFF; /* Maximum timeout value */
    WDOG->CS = 0x00002100; /* Disable watchdog */
    }
    void Enable_Interrupt(uint8_t vector_number)
    {
    S32_NVIC->ISER[(uint32_t)(vector_number) >> 5U] = (uint32_t)(1U << ((uint32_t)(vector_number) & (uint32_t)0x1FU));
    S32_NVIC->ICPR[(uint32_t)(vector_number) >> 5U] = (uint32_t)(1U << ((uint32_t)(vector_number) & (uint32_t)0x1FU));
    }
    int main(void)
    {
    /* Initialization
    * ================
    */
    WDOG_disable(); /* Disable WDOG */
    PORT_init(); /* Configure ports */
    SOSC_init_8MHz(); /* Initialize system oscilator for 8 MHz xtal */
    SPLL_init_160MHz(); /* Initialize SPLL to 160 MHz with 8 MHz SOSC */
    NormalRUNmode_80MHz(); /* Init clocks: 80 MHz sysclk & core, 40 MHz bus, 20 MHz flash */
    EWM_init(); /* Initialize EWM. */
    Enable_Interrupt(WDOG_EWM_IRQn); /* Enable EWM interrupt. */
    /* Wait forever
    * ============
    */
    for(;;)
    {
    }
    }
