void EWM_init ( void  )

EWM initialization:
Prescaled clock frequency =
low power clock source frequency / ( 1 + CLK_DIV ) =
LPO 128KHz / 128 = 1KHz.
The EWM compare registers are used to create a
refresh window to refresh the EWM module.
Refresh window is [15ms - 255ms].
EWM Interrupt enabled.
