◆ PTC6

#define PTC6   (6)

Description:

This example shows how the WDOG works in Window Mode. A button refreshes the WDOG manually. If the button is pushed too early or too late, the WDOG resets the MCU. Once the WDOG is refreshed in the allowed time, the WDOG counter (CNT) starts again. You must refresh the WDOG periodically to avoid the MCU reset. For this example, a valid refresh is between 5 and 10 seconds from when the WDOG counter started running. This values can be changed in the WIN and TOVAL registers respectively. The UART is only used to display the WDOG status in the terminal at 115200 baud: TeraTerm or other software.
